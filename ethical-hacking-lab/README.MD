# Ethical Hacking Lab – Kali Linux & Metasploitable

## Descripción

Implementación de un laboratorio de hacking ético en entorno virtual para pruebas controladas.

## Objetivo

Configurar una red cerrada entre Kali Linux y Metasploitable2 para realizar pruebas de seguridad.

## Arquitectura

- Kali Linux (Atacante)
- Metasploitable2 (Víctima)
- Red interna aislada

## Configuración

- Red tipo Host-Only / Interna
- Asignación de IP manual
- Verificación de conectividad mediante ping

## Pruebas realizadas

- Conectividad entre máquinas
- Acceso al servicio web de Metasploitable

## Resultado

Se logró establecer comunicación exitosa entre ambas máquinas y acceso al entorno web vulnerable.
